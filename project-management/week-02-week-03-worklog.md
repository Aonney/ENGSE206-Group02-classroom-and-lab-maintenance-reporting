## บันทึกการทำงานรายบุคคล (Individual Logs)

### 1. นายสันติ ปัญญาหน้อย (บทบาท: Project Setup & Documentation)
- **สิ่งที่ทำในสัปดาห์นี้:**
  - ตั้งค่าสิทธิ์การเข้าถึง Git Repository (Collaborator) ให้สมาชิกในกลุ่มสามารถ Push งานร่วมกันได้
  - จัดเตรียมและเขียนข้อมูลเป้าหมายโจทย์เบื้องต้นลงในเอกสาร `CASE_CARD.md` และ `TEAM.md`
  - ทำการเรียบเรียงข้อมูลสเตกโฮลเดอร์และขอบเขตโครงงานในไฟล์ `docs/02-stakeholder-context-scope.md`
- **สถานะงาน:** เสร็จสิ้น (Done)

### 2. นายธีรนัย ไชยกันทะ  (บทบาท: System Modeling & Diagrams)
- **สิ่งที่ทำในสัปดาห์นี้:**
  - ร่วมระดมสมองคัดกรองรายชื่อผู้มีส่วนได้ส่วนเสียและความต้องการของระบบร่วมกับทีม
  - ออกแบบและวาด **System Context Diagram** เพื่อกำหนดขอบเขตข้อมูลเข้า-ออกของระบบรายงานการซ่อมแซม
  - ออกแบบและวาด **Stakeholder Map** (Power vs Interest Grid) เพื่อจัดกลุ่มผู้เกี่ยวข้องทั้ง 4 กลุ่มตามหลักการระบบ
  - จัดการโครงสร้างโฟลเดอร์ `diagrams/context/` และสร้างโฟลเดอร์ `diagrams/stakeholders/` บน GitHub พร้อมอัปโหลดไฟล์ `.drawio` และ `.png`
- **สถานะงาน:** เสร็จสิ้น (Done)

### 3. นายบูรพา ประทีปรัตน์  (บทบาท: Quality Assurance & Peer Feedback)
- **สิ่งที่ทำในสัปดาห์นี้:**
  - ร่วมวิเคราะห์พฤติกรรมและความต้องการของผู้ใช้งานห้องเรียนและเจ้าหน้าที่เทคนิค
  - ทำการตรวจสอบความถูกต้องของสโคปงานและเนื้อหาเอกสารทั้งหมดภายในกลุ่ม
  - ดำเนินการเขียนแบบประเมินและข้อเสนอแนะให้กลุ่มอื่นในไฟล์ `feedback/week-02-peer-feedback.md`
- **สถานะงาน:** กำลังดำเนินการ/เสร็จสิ้น (In Progress / Done)
